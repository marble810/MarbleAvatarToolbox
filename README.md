# Marble Avatar Tools

A Unity package containing custom Editor Scripts specifically designed for VRChat Avatar editing and optimization.

## Features

- **PhysBone Collider Batch Setup**: Streamline the process of adding and configuring PhysBone colliders in VRChat avatars
- **Avatar Optimization Tools**: Various utilities for optimizing avatar performance
- **Editor Automation**: Tools to automate repetitive tasks in avatar creation workflow

## Requirements

- Unity 2019.4 or higher
- VRChat SDK 3.0 or higher

## Installation

1. Open your Unity project
2. Open the Package Manager (Window > Package Manager)
3. Click the "+" button and select "Add package from disk..."
4. Navigate to this package folder and select the `package.json` file

## Usage

### PhysBone Collider Batch Setup

The PhysBone Collider Batch Setup tool allows you to quickly configure PhysBone colliders across your avatar:

1. Open the tool from the Unity Editor menu
2. Select your avatar GameObject
3. Configure batch settings
4. Apply colliders to selected bones

## Development

All editor scripts are located in `Assets/MarbleScripts/Editor/` and follow Unity's editor scripting best practices.

## Compatibility

- VRChat SDK 3.0+
- Unity 2019.4+
- Compatible with Modular Avatar, Avatar Optimizer, and other popular VRChat avatar tools
