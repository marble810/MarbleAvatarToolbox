# PhysBone Collider Batch Setup Sample

This sample demonstrates how to use the PhysBone Collider Batch Setup tool.

## Contents

- Example avatar prefab with PhysBone components
- Sample collider configurations
- Usage documentation

## Setup Instructions

1. Import this sample into your project
2. Open the example scene
3. Select the example avatar
4. Use the PhysBone Collider Batch Setup tool to configure colliders

## Features Demonstrated

- Batch collider creation
- Automatic bone detection
- Collider parameter presets
- Performance optimization tips

For more information, see the main package documentation.
